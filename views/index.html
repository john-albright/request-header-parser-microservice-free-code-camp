Request Header Parser Microservice

Project Description

This is project 2 of 5 for FreeCamp.org's APIs and Microservices certificate, the 5th of 6 full stack JavaScript certificates. This API parses some of the information contained in the header of a request: the IP address of the computer, the language of the browser, and the browser and computer being used to make a request.

Example Usage

The subdirectory /api/whoami is the only one that can be accessed.

Example Output

This is the output returned on my computer:

{"ipaddress":"35.191.15.131", "language":"es-ES,es;q=0.9","software":"Mozilla/5.0 (Macintosh; Intel Mac OS X 10_11_6) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/92.0.4515.131 Safari/537.36"}
